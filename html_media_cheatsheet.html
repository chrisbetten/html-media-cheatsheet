HTML Media Reference

Image

Image Tag

Tag Name Description
<img>     Image       Embed image in an HTML page. Self-closing.   

Common Image Attributes

Attribute   Name Description
src Source Specifies the path to the image.
alt Alternate    Specifies an alternate text for an image.
height Height Specifies the height of an image.
width Width Specifies the width of an image.

Video

Video Tags

Tag Name Description
<video> Video The wrapper element for all video content.
<source>    Source     Contains source of video content. Self-closing.

Supported video formats in HTML: MP4, WebM, and OGG.

Common Video Attributes

Attribute Name Description
autoplay   Autoplay      Specifies that the video will start playing as soon as it is ready.
controls Controls Specifies that video controls should be displayed (such as a play/pause button etc).
height Height Sets the height of the video player.
loop Loop Specifies that the video will start over again, every time it is finished.
muted Muted Specifies that the audio output of the video should be muted.
src Source Specifies the URL of the video file.
width Width Sets the width of the video player.

Audio

Audio Tags

Tag Name Description
<audio> Audio The wrapper element for embedding sound content, e.g. music or other audio streams.
<source>      Source     Contains source of audio content. Self-closing.

Supported audio formats in HTML: MP3, WAV, and OGG.

Common Audio Attributes

Attribute Name Description
autoplay      Autoplay      Specifies that the audio will start playing as soon as it is ready.
controls Controls Specifies that audio controls should be displayed (such as a play/pause button etc).
loop Loop Specifies that the audio will start over again, every time it is finished.
muted Muted Specifies that the audio output should be muted.
src Source Specifies the URL of the audio file.
